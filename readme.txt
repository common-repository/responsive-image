== Description ==
Responsive Image – Bringing automatic default responsive images to your content.
The image on your content, post, pages, and etc will be automatically responsive.

Plugin Features: 

1. Content Auto Responsive.
2. all devices the image on your post, pages etc. will be responsive.
3. Its easy to use.
4. Tested up to latest Wordpress

== Installation ==

This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire "Responsive Image" folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done
